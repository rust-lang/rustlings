# Rust 新特性练习

这个章节包含了一些 Rust 最新特性的练习。通过这些练习，你将学习到：

1. const 泛型（Rust 1.51+）
2. GAT (Generic Associated Types) (Rust 1.65+)
3. let-else 语句（Rust 1.65+）
4. 异步特征（Rust 1.75+）
