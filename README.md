# [Rustlings](https://rustlings.rust-lang.org) 🦀

Small exercises to get you used to reading and writing [Rust](https://www.rust-lang.org) code - _Recommended in parallel to reading [the official Rust book](https://doc.rust-lang.org/book) 📚️_

Visit the **website** for a demo, info about setup and more:

## ➡️ [rustlings.rust-lang.org](https://rustlings.rust-lang.org) ⬅️
## Quick Start

If you are returning to Rustlings, you can simply run:

rustlings watch

To set up Rustlings for the first time:

1. Install Rust:
   https://www.rust-lang.org/tools/install

2. Clone this repository:
   git clone https://github.com/rust-lang/rustlings
   cd rustlings

3. Install Rustlings:
   cargo install --path .

4. Start Rustlings:
   rustlings watch

This command will automatically run and check your exercises as you work.
