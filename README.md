##  Getting Started

Run these commands to get started with Rustlings:

```bash
# 1. Verify Rust is installed
rustc --version

# 2. Clone the repository
git clone https://github.com/rust-lang/rustlings.git
cd rustlings

# 3. Install Rustlings
cargo install --path .

# 4. Start Rustlings
rustlings watch
